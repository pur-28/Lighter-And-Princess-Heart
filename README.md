A charming, interactive web experience designed to spark emotion and curiosity.
Just click to begin and let the magic unfold ✨

## 🔗 Live Demo

👉 https://pur-28.github.io/Lighter-And-Princess-Heart/

---

## ✨ Features

* 🎯 One-click interactive start
* ❤️ Heart-themed visual storytelling
* 🎶 Background music for immersive feel
* 🌈 Smooth and engaging user experience

---

## 🚀 How to Use

1. Open the live site
2. Click the **"Click to Start"** button
3. Follow the on-screen prompts and enjoy the experience

---

## 🛠️ Run Locally

```bash
git clone https://github.com/pur-28/Lighter-And-Princess-Heart.git
cd Lighter-And-Princess-Heart
```

Then simply open `index.html` in your browser
(or use a local server for best performance)
